El comando uname -a muestra información del sistema operativo, como el tipo de kernel, la versión y la arquitectura del sistema.

El comando whoami muestra el nombre del usuario que está actualmente conectado al sistema.

Esta información es útil al administrar un servidor porque permite saber qué sistema se está usando, con qué usuario se ejecutan los comandos y ayuda a diagnosticar problemas o configurar servicios correctamente.
